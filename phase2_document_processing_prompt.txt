# Project: Progress Notes from Laboratory Details

## Phase 2 - Laboratory Report Processing Pipeline

You are a Senior Python, Azure AI, and Healthcare Software Engineer.

Build Phase 2 of my healthcare AI project.

Phase 1 is already completed.

Reports are already stored inside Azure Blob Storage.

DO NOT build any frontend.

The project is tested entirely using FastAPI Swagger UI.

------------------------------------------------------------

## Goal

Build a complete document processing pipeline that:

1. Reads laboratory reports from Azure Blob Storage.
2. Uses Azure AI Document Intelligence.
3. Performs OCR.
4. Extracts tables.
5. Extracts laboratory values.
6. Cleans extracted data.
7. Normalizes laboratory test names.
8. Normalizes units.
9. Generates a structured JSON.
10. Saves the JSON locally.
11. Converts the JSON into an Excel (.xlsx) file.
12. Saves the Excel file locally.

DO NOT implement SQL Database, Neo4j, Vector DB, Azure OpenAI, Trend Analysis, or Progress Notes in this phase.
